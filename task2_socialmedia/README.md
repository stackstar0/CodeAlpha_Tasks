# Social Media Platform

A simple social media application built with Node.js, Express, and Vanilla JS.

## Features
- **Login**: Mock authentication system.
- **Feed**: View posts, create new posts, like, and comment.
- **Profile**: View user profiles, follow/unfollow users.
- **Responsive Design**: Modern UI with glassmorphism aesthetics.

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Node.js, Express.js
- **Database**: In-memory (users/posts reset on server restart)

## Setup & Run

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Server**
   ```bash
   node server.js
   # or with nodemon
   npx nodemon server.js
   ```

3. **Open Application**
   Visit [http://localhost:3000](http://localhost:3000) in your browser.
